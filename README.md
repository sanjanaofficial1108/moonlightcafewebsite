#  Moonlight Café Website

A modern, responsive ** Café website** built using **HTML, CSS, JavaScript, and PHP**.  
This project showcases both **front-end UI/UX design** and **basic back-end email handling**.

---

##  Features

-  Cinematic night-style café design
-  Glassmorphism navigation bar
-  Animated hero banner with background image
-  Menu section with images & hover effects
-  Floating feedback panel with star rating
-  Contact form with email functionality (PHPMailer)
-  Google Map location embed
-  Fully responsive layout
